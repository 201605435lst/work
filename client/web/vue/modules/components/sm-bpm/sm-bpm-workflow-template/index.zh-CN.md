## API

| 参数  | 说明                         | 类型     | 默认值 |
| ----- | ---------------------------- | -------- | ------ |
| axios | 项目 axios.create 创建的实例 | function |
| single | 是否为单一流程模式 | boolean | false|

## 方法

| 名称        | 描述                                    | 参数       |
| ----------- | --------------------------------------- | --------------- |
| view()        | 查看事件                                 | (id) => void    |
| edit()        | 编辑事件                                 | (id) => void    |
